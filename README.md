# Bridge_Science

**橋梁科学プロジェクト**  
このリポジトリでは、橋梁に関するデータの収集・分析・モデル化に取り組んでいます。  
Python、Jupyter、GitHubを用いたデータサイエンス・機械学習の実験環境を構築しています。

---

## 🚀 特徴・目的

- 橋梁の属性データをもとに分析・可視化
- データ前処理と簡易モデルの作成
- 将来的にはAIによる健全度予測モデルの開発を目指す

---

## 🧰 使用技術・環境

- Python 3.10+
- pandas / numpy / matplotlib / folium
- Jupyter Notebook
- GitHub + Docker（予定）
